# 🥂 TL Rosendos - Eventos & Buffet

Uma landing page sofisticada desenvolvida para a marca **TL Rosendos**, focada em converter visitantes em clientes através de uma interface clara e visualmente impactante. O projeto destaca dois braços de atuação: Assessoria de Eventos e Serviços de Gastronomia/Buffet.

## 🎨 Design & Conceito
O projeto utiliza um contraste visual estratégico para separar os serviços:
- **Assessoria:** Estilo *Clean & Chic* com paleta Branco Gelo e Preto, utilizando fontes serifadas modernas.
- **Buffet:** Estilo *Gourmet & Dark* com paleta Preto e Ouro Velho (Amarelo Queimado), transmitindo exclusividade.

### Tipografia utilizada (Google Fonts):
- **Títulos:** `Bodoni Moda` (Serifada clássica de alto contraste).
- **Destaques:** `Mea Culpa` (Caligrafia fluida/Handwriting).
- **Corpo:** `Montserrat` (Sans-serif moderna para legibilidade).

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estrutura semântica.
- **CSS3:** Layouts com Flexbox, Grid e animações de transição.
- **JavaScript (Vanilla):** Lógica para menu ativo durante o scroll e interatividade básica.
- **Mobile First:** Totalmente responsivo para dispositivos móveis.
